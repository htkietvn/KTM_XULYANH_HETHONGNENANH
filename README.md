# Hệ Thống Nén Ảnh

Web application đơn giản để nén và so sánh ảnh với nhiều định dạng khác nhau.

## Tính năng

- **Upload ảnh**: Hỗ trợ các định dạng ảnh phổ biến
- **Nén ảnh**: 
  - JPEG (Lossy) - Điều chỉnh chất lượng 0-100%
  - PNG (Lossless) - Giữ nguyên chất lượng
  - WebP - Định dạng hiện đại, kích thước nhỏ
- **So sánh**: Xem trước ảnh gốc và ảnh đã nén cạnh nhau
- **Phân tích**: Thống kê tỷ lệ nén, dung lượng tiết kiệm
- **Tải xuống**: Lưu ảnh đã nén về máy



## Cách sử dụng

1. Mở file `index.html` trong trình duyệt
2. Click "Chọn file" và tải ảnh lên
3. Chọn định dạng nén và điều chỉnh chất lượng
4. Click "Nén ảnh" để xem kết quả
5. So sánh và tải xuống nếu hài lòng

## Công nghệ

- HTML5 Canvas API
- JavaScript (Vanilla)
- CSS3
